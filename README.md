This project is a fully responsive, Single Page Application (SPA) designed to showcase the features of Helios, an all-in-one Discord bot. Unlike standard clean corporate sites, this project embraces a "Sketchy/Doodle" design language, utilizing hand-drawn fonts, jittery borders, and paper textures to create a unique user experience.

Despite the playful look, the site is powered by heavy-duty frontend animation libraries to ensure a premium, smooth feel.

Single File Architecture: The entire application (Routing, Logic, Styles, Content) is contained within a single index.html file for easy portability.

SPA Routing: Custom vanilla JavaScript router handles navigation between Home, Features, Commands, and Legal pages without page reloads.

Mobile Optimized: Fully responsive layout with touch-optimized interactions for the canvas background and mini-game.

üé¨ Animations & VFX

"Rabbit Smash" Intro: A custom SVG animation sequence where a hammer smashes a rabbit to reveal the website, complete with screen shake and impact effects.

3D Smooth Scroll: Integrated Lenis scrolling with a custom velocity-based skew and rotation effect for a 3D rolling feel.

Magnetic Buttons: Buttons magnetically pull towards the cursor on desktop using GSAP.

3D Tilt Cards: Feature cards tilt in 3D space based on mouse position.

Parallax Hero: Floating elements in the hero section react to mouse movement.
